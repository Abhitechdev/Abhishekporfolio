## ✨ Features & User Experience (UX)

### 🚀 Modern Split-Column Brand Hero
* Features my official **AbhiTech Decoded** profile logo and microprocessor brand theme.
* Balanced split-column layout on desktop to display the portrait perfectly from head to chest without any cropping.
* Sleek linear gradient overlays (`hero-img-glow`) that seamlessly blend the profile picture into the glassmorphic content cards.

### 🎯 Custom Trailing Physics Cursor
* **Fluid Spring Inertia**: Custom two-part cursor consisting of a laser-accurate dot and an outer trailing ring built with spring physics using `requestAnimationFrame` for a smooth, high-fidelity liquid feel.
* **Hover Expansion**: The central dot vanishes and the trailing ring dynamically scales up, filling with a warm glow whenever hovering over interactive cards, buttons, or links.
* **Click feedback**: Instant visual compression on click to provide tactile confirmation.

### 💡 Mouse-Tracking Volumetric Spotlight Cards
* **Shining Moving Borders**: High-performance CSS masking (`-webkit-mask-composite: xor`) draws a dynamic, golden shining border on skill cards and contact nodes that literally flows along the border edge following your cursor.
* **Ambient Glows**: Subtle radial gradient spotlights light up behind cards on hover, creating beautiful volumetric depth.

### 🎬 Featured Reels Player
* Seamless integration of top featured MP4 tech reels.
* Clean custom interaction overlay: clicking a card plays the reel with audio, automatically pauses/mutes all other playing videos, and swaps out play/pause state icons gracefully.

## 🛠️ Built With

* **Markup**: Semantic HTML5 structures.
* **Styling**: Vanilla CSS3 utilizing Custom Properties (Variables), advanced CSS mask systems, Keyframe Animations, Glassmorphism, and HSL palettes.
* **Logic**: Pure Vanilla ES6 JavaScript (zero dependencies) driving custom cursor physics, Intersection Observer scroll animations, tab state machines, and parallax scrolls.

## ⚡ How to Run Locally

You don't need any complex build steps or dependencies to run this portfolio! 

### Option 1: Double-Click (Zero Setup)
Simply navigate to your local clone directory and double-click `index.html` to open it instantly in any modern web browser.

### Option 2: Live Local Dev Server
For the best experience (including optimized video stream caching), run a local server:
1. Open terminal inside the project directory:
   ```bash
   npx serve -l 3000
   ```
2. Open your browser and navigate to:
   ```text
   http://localhost:3000
   ```
